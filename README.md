# 🍽️ Vite & Gourmand

Application web de gestion de menus et commandes pour un service traiteur.

## 🚀 Installation

1. Cloner le projet
2. Placer le dossier dans `htdocs` (XAMPP)
3. Démarrer Apache et MySQL
4. Importer la base de données via phpMyAdmin avec le fichier `database.sql`
5. Accéder au site :
http://localhost/vite-gourmand
